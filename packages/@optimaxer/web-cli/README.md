# Optimaxer CLI


# Command and Action Generator

### Installation
`
npm install -g optimaxer-cli
`

### Example Usage
`
optimaxer-cli gen -cf <Config file> -tf <Training Factor> -oak <Open AI Key>
`

- **`-cf`** : Config file path (JSON). Example: `config.json`
- **`-tf`** : Training Factor
- **`-oak`** : Open Ai Key

### Help Doc

`
optimaxer-cli help gen
`
